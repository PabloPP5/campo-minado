# Campo Minado em C

Mini-projeto desenvolvido em C com o objetivo de praticar o uso de matrizes, funções e lógica de programação.

O programa implementa uma versão simples do jogo Campo Minado no terminal, onde o usuário escolhe posições da matriz e o sistema informa a quantidade de bombas ao redor ou encerra o jogo em caso de explosão.

## Funcionalidades

- Geração do campo minado em uma matriz 10x10  
- Posicionamento aleatório das bombas  
- Cálculo da quantidade de bombas vizinhas  
- Abertura das casas escolhidas pelo usuário  
- Encerramento do jogo ao selecionar uma bomba  

## Objetivo do projeto

Praticar lógica de programação, principalmente o uso de matrizes, estruturas de repetição, funções e manipulação de índices em C.

## Como executar

Compile e execute o arquivo principal:

```bash
gcc campominado.c -o campominado
./campominado
```
